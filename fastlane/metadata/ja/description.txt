Web 上や他アプリで保存済みの画像を保存、管理できるシンプルなアプリです。


Web サイトの URL や画像を、他アプリ (ブラウザや Twitter 等のSNS 、写真アプリなど) の「共有」から、LikePics へシェアしてください。

Web サイトを共有すると、Web サイト上の画像を抽出してダウンロードすることができます。この時、画像データのみでなく、ダウンロード元の Web サイトの URL も保存できます。これにより、アプリ内に保存した画像から、保存元の Web サイトをすぐに開くことができます。

また、画像をクリップやアルバムにまとめたり、任意のタグを付与して管理することができます。

保存・管理しているデータは iCloud により、複数の iOS/iPad デバイスで共有できます。
